Hints for developers:
=====================
1. 
This demo project was developed and tested with the Eclipse Versions 3.6.2 (Helios) 
and Version 3.5.1 (Galileo) for the Android Api Level 7 ( Revision 2.1 ).
The related Android SDK version is 15.

2. 
You need the "SDCFramework" Project in the same workspace to build it,
as this project uses libraries from the framework project.
