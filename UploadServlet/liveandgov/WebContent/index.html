Live+Gov Data Collection App


Installation Instructions

To install the mobile app on your Android device please:
  1. Enable Software installation from Unknown Sources
    Settings > Security > Check: Unknown Sources [x]
  2. Open the following link on your with your phone browser:
    Download liveandgov.apk or http://goo.gl/IZA7P or
  3. The Live+Gov Data Collection App with reduced sensor load:
    Download liveandgov2.apk or http://goo.gl/WQhlU or
Please uninstall any previously installed version beforehand.

Usage

Please record sensor data from your journeys e.g. way to work.
  • Start the SDCF service and Sample Recordning when you start your journey
  • Use the dropdown menu to specify your means of travel (whenever possible)
  • Use the free text to add further specification e.g. service line (whenever possible)
  • Stop recording when you arrive at work.
Leave the service running for a few more minutes after the recording is finished. The App will upload the recorded samples as soon as a wifi connection is available.

Andorids Rights

The app requires a number of rights which are listed below. Please note, that this app will not (!) record sensible information like your phone number, calls, etc. nor will this app modify existing contents of your SD card.
  • read phone state and identity - for recording visible GSM-cells (not Phone number, or calls!)
  • approximate/precise location - for recording location
  • modify or delete the contents of your USB storage - save samples to sd card
  • change network connectivity - enable wifi for recording
  • connect and disconnect from Wi-Fi - enable wifi for recording
  • full network access - transfer samples to server
  • view network connections - record visible Wi-Fi networks
  • access Bluetooth settings - record bluetooth samples
  • pair with bluetooth devices - not needed
  • control vibration - for notifications
  • prevent phone from sleeping - keep alive while recording samples
  • mock location sources for testing - not needed
  • modify system settings - sore application settings
  • read battery statistics - monitor battery state
  • test access to protected storage - not needed
